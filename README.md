# NFL DSC Project

A data-driven NFL analytics project built for the NYU DSC Fall 2025 Project Exposition, with the goal of predicting the Super Bowl champion. 
Includes data cleaning, feature engineering, boosted trees, and logistic regression models to predict game outcomes.
